Scope and objectives:

This course is aimed at people who have little to no R experience, we will open with an Introduction to R introducing the fundamental features of the language. This will serve as a foundation for the rest of the course.

Next up: Data Wrangling! Here you'll become more familiar with your new best friend, the Tidyverse. From loading data from different file types to functional programming with purrr, we've got it all covered.

On day 2 we'll demonstrate how to combine the basic components of ggplot2 in order to form sophisticated visualisations. Weâ€™ll then take a look at extensions for labelling points in congested scatterplots and producing animations.

Machine Learning will be our next focus. We'll cover both classification and linear models.

By the end of day 4 you'll be well on your way to becoming a Shiny sovereign having learned how to:

  • understand the structure of a Shiny application (UI and server)
  • assemble an attractive UI
  • understand reactivity and how the UI and server communicate with each other
  • deploy a Shiny application

Our final day, dedicated to RMarkdown & automated reporting, will teach you everything you need to know in order to source & prepare data, create an R markdown doc with tables and figures and how to schedule recurring reports.

Highly-practical & interactive

Our training emphasises practical skills. So, although you'll be learning concepts and theory, you'll see how everything is applied in the real world as we work through examples and exercises based on real datasets. We like questions! Having a firm understanding of the content of the course will result in you being able to confidently apply your new skills thereafter so if at any point you're unsure of something ASK!

Requirements:

All you'll need is a computer with a browser. We'll be using an online development environment. This means that we can focus on learning as much as possible without having to dedicate time to solving technical problems.

Of course, we are happy to help you get your local environment set up too! You can start by following these instructions.
